# crazylxr`s blog
这是一个基于 github page 和 github api 搭建的单页面静态博客。

## 环境要求
- nodejs 8.0 以上

查看node版本方法

```
node -v
```
如果版本低于8.0，采用n模块升级，步骤如下
1. 安装n模块

```
npm install -g n
```
2. 升级到稳定版本

```
n stable
```
升级完成之后记得重新查看是否升级成功

## 快速开始
### 安装依赖

```
npm install
```
### 开发

```
npm run dev
```
然后访问localhost:1234

### 打包

```
npm run build
```
